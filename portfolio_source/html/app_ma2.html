프로젝트 설명

모바일에서 증권 거래가 가능한 애플리케이션입니다.

역할

퍼블리싱 100% 사용자의 시선을 끌 수 있도록 애니메이션으로 콘텐츠에 동적인 효과를 주어 작업하였고, 검색창에 입력된 문구 중 공통 문구에 반응하는 검색창을 구현하였습니다. 모바일 디바이스를 기반으로, VW 단위를 사용하여 작업하여 어떤 디바이스에서 접속하더라도 동일한 화면을 보여줄 수 있도록 작업하였습니다. 작업 중 기획자, 디자이너, 개발자와의 소통을 통한 의견 조율, 문제 해결을 경험하였습니다.
